##################### steps to create data model ####################

rails generate model User name:string photo:string bio:text posts_counter:integer

rails generate model Post title:string text:text author:references comments_counter:integer likes_counter:integer

rails generate model Like author:references post:references

rails generate model Comment author:references post:references text:text



############### create models #######################

rails generate model User name:string photo:string bio:text posts_counter:integer
rails generate model Post author_id:bigint title:string text:text comments_counter:integer likes_counter:integer
rails generate model Comment author_id:bigint post_id:bigint text:text
rails generate model Like author_id:bigint post_id:bigint
